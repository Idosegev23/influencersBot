## 🧭 ניווט במערכת

לאחר ההתחברות, תראה **תפריט ניווט עליון** עם הכפתורים הבאים:

| אייקון | כפתור | מה תמצא |
|--------|--------|---------|
| 🏠 | **דשבורד** | סקירה כללית, KPIs, משימות קרובות |
| 🤝 | **שת"פים** | כל השת"פים שלך - פעילים/הסתיימו |
| ✅ | **משימות** | רשימת משימות, סינון, עדכון סטטוס |
| 🎫 | **קופונים** | ביצועי קופונים, המרות, הכנסות |
| 💬 | **תקשורת** | תקשורת עם מותגים לפי קטגוריות |
| 👥 | **קהל** | נתוני פעילות קהל, שיחות, המרות |
| 📄 | **מסמכים** | כל המסמכים שהעלית |

## 📋 תהליכים עיקריים

### 1️⃣ יצירת שת"פ חדש

#### אפשרות א: ידנית
1. לחץ על **שת"פים** בתפריט
2. לחץ **+ הוסף שת"פ חדש**
3. מלא את הפרטים:
   - שם המותג *
   - שם הקמפיין
   - סטטוס (Lead, משא ומתן, פעיל, וכו')
   - תאריכי התחלה/סיום
   - סכום החוזה
   - דליברבלס
   - הערות
4. לחץ **צור שת"פ**

#### אפשרות ב: מהעלאת מסמך (מומלץ! 🤖)
1. צור שת"פ ריק או בחר קיים
2. לחץ על טאב **מסמכים**
3. לחץ **העלה מסמך**
4. בחר PDF/Word עם הצעת מחיר או חוזה
5. **ה-AI מנתח אוטומטית!** ⏳
6. לחץ **סקור נתונים**
7. ערוך/תקן את מה שה-AI מצא
8. לחץ **צור שת"פ מהמסמך**
9. **השת"פ מלא ומוכן!** 🎉

---

### 2️⃣ ניהול מסמכים

#### העלאת מסמך:
- **דרך שת"פ:** `/partnerships/[id]` → טאב "מסמכים"
- **סוגי קבצים נתמכים:** PDF, Word (.doc/.docx), תמונות (JPG/PNG)
- **גודל מקסימלי:** 50MB

#### מה ה-AI מנתח:
- 📄 **הצעת מחיר (Quote):**
  - שם מותג
  - שם קמפיין
  - סכום כולל
  - דליברבלס (פוסטים, סטוריז, רילס)
  - תאריכים
  - תנאי תשלום

- 📝 **חוזה (Contract):**
  - צדדים (מותג, משפיען, סוכן)
  - מספר חוזה
  - תאריכים (חתימה, תחילה, סיום)
  - בלעדיות
  - תנאי תשלום
  - סעיפי ביטול

- 📋 **בריף (Brief):**
  - מטרת הקמפיין
  - קהל יעד
  - הודעות מפתח
  - Hashtags & Mentions
  - הנחיות תוכן
  - משימות + deadlines

#### Confidence Score:
- 🟢 **80%+ (ירוק):** מדויק מאוד, בדוק רק בסיסי
- 🟡 **60-80% (צהוב):** די טוב, בדוק היטב
- 🔴 **<60% (אדום):** בדוק ותקן הכל

---

### 3️⃣ ניהול משימות

#### יצירת משימה:
1. לחץ **משימות** → **+ משימה חדשה**
2. מלא:
   - כותרת
   - תיאור
   - סוג (פוסט, סטורי, ריל, meeting, וכו')
   - סטטוס
   - עדיפות
   - תאריך יעד
   - שייכות לשת"פ (אופציונלי)

#### עדכון משימה:
- **מהירה:** לחץ על המשימה → **פעולות מהירות**
  - התחל משימה
  - סמן כהושלם
  - סמן כחסום
- **מלאה:** לחץ **ערוך** → שנה פרטים → **שמור**

---

### 4️⃣ תקשורת מותגים

#### קטגוריות:
- 💰 **פיננסי** - תשלומים, חשבוניות, תמחור
- ⚖️ **משפטי** - חוזים, זכויות, תנאים
- 🚨 **בעיות שת"פ** - תלונות, בעיות ביצוע
- 📢 **כללי** - שאר התקשורת

#### סטטוסים:
- ✅ **פתוח** - דורש מעקב והתראות
- ⏹️ **סגור** - טופל והסתיים

#### יצירת תקשורת:
1. **תקשורת** → **+ תקשורת חדשה**
2. בחר מותג
3. כותרת + תיאור
4. קטגוריה + עדיפות
5. **שמור** → התקשורת נוצרת + התראות אוטומטיות!

### 5️⃣ אנליטיקס וביצועים

#### דשבורד קהל (👥 Audience):
- סה"כ שיחות (בתהליך / נסגרו)
- הודעות ממוצע לשיחה
- קופונים הועתקו/נוצלו
- אחוז המרה (שיחה → קופון)
- שביעות רצון
- **גרף צמיחה** - שיחות לאורך זמן

#### אנליטיקס קופונים (🎫 Coupons):
- סה"כ קופונים
- כמה הועתק / שומש
- הכנסות בפועל (₪)
- אחוז המרה (העתקה → שימוש)
- **עוקבים vs לא עוקבים**
- **המוצרים הנמכרים ביותר**
- רווח פר קופון
- סל ממוצע

#### ROI Calculator:
בתוך שת"פ בודד → טאב **ROI**
- השקעת זמן
- הכנסות בפועל
- רווח נטו
- החזר השקעה (ROI %)

## 🤖 AI Features

### Document Parsing (Gemini Vision):
- **Upload** → ממתין
- **Processing** → AI מנתח (5-30 שניות)
- **Completed** → נתונים מלאים
- **Failed** → מילוי ידני

### Confidence Indicators:
```
🟢 80-100% = מצוין
🟡 60-79% = טוב
🔴 0-59% = בדוק היטב
```

### מה ה-AI יכול לחלץ:
- ✅ שמות מותגים וקמפיינים
- ✅ סכומים ומטבעות
- ✅ תאריכים (התחלה, סיום, deadlines)
- ✅ דליברבלס (כמות, סוג, פלטפורמה)
- ✅ תנאי תשלום (milestones, אחוזים)
- ✅ אנשי קשר (שם, מייל, טלפון)
- ✅ סעיפים מיוחדים

## 🎯 Workflows מומלצים

### Workflow 1: שת"פ חדש עם מסמך
```
1. קיבלתי הצעת מחיר במייל
2. שמרתי PDF
3. נכנסתי לשת"פים → + חדש
4. יצרתי שת"פ ריק
5. טאב מסמכים → העלאת PDF
6. AI ניתח תוך 20 שניות
7. סקרתי + תיקנתי פרט אחד
8. השת"פ מלא ומוכן! ⚡
```

**חיסכון:** 15 דקות של הקלדה ידנית!

### Workflow 2: מעקב אחרי שת"פ פעיל
```
1. נכנסתי לשת"פים
2. בחרתי שת"פ פעיל
3. צפיתי בפרטים
4. טאב מסמכים → יש חוזה + בריף
5. משימות אוטומטיות נוצרו מה-brief
6. קיבלתי התראה על deadline מתקרב
7. סימנתי משימה כהושלמה
8. העליתי חשבונית
```

### Workflow 3: בדיקת ביצועים
```
1. קהל → ראיתי סה"כ שיחות
2. קופונים → ראיתי המרות
3. לחצתי על קופון ספציפי
4. ראיתי אילו מוצרים נמכרו
5. חישבתי ROI
6. החלטתי על upsell
```

- 🟠 **High** - עד 3 ימים
- 🟡 **Medium** - עד שבוע
